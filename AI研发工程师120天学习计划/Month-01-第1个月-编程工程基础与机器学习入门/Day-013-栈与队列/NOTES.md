## 核心笔记

- 栈的核心特征是后进先出，适合处理“最近的状态”。
- 当题目和括号匹配、撤销/回退、最近未处理元素有关时，优先考虑栈。
- 有效括号题中，左括号入栈，右括号检查栈顶是否匹配，全部处理完后栈必须为空。
- 单调栈常用于“右边/左边第一个更大元素”“右边/左边第一个更小元素”这类题。
- 单调栈里经常存的是“下标”而不是“值”，因为下标不仅能取到值，还方便直接给结果数组赋值。
- 对单调栈来说，题目的关键词往往是：最近满足单调关系的元素，而不只是泛泛的“就近匹配”。

## 代码/实验结果

- 有效括号代码：

```python
def is_valid(s):
    stack = []
    pairs = {
        ')': '(',
        ']': '[',
        '}': '{'
    }

    for ch in s :
        if ch in pairs:
            if not stack or stack[-1] !=pairs[ch]:
                return False
            stack.pop()
        else:
            stack.append(ch)

    return len(stack) == 0

print(is_valid("()[]{}"))
print(is_valid("(]"))
print(is_valid("([{}])"))
```

- 输出结果：

```text
True
False
True
```

- 下一个更大元素代码：

```python
def next_greater(nums):
    result = [-1] * len(nums)
    stack = []
    for i in range(len(nums)):
        while stack and nums[i] > nums[stack[-1]]:
            index = stack.pop()
            result[index] = nums[i]
        
        stack.append(i)

    return result

nums = [2, 1, 2, 4, 3]
print(next_greater(nums))
```

- 输出结果：

```text
[4, 2, 4, -1, -1]
```

- 结果说明：
  已完成普通栈和单调栈的入门题，能够看懂“最近状态”和“最近更大元素”两种典型栈逻辑。
