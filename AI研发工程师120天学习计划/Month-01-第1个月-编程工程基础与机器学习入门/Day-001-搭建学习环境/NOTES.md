## 代码/实验结果

- 编写代码：

```python
name = "hka"
days = 120
print(f"my name is{name}")
print(f"try to learn {days}天")
```

- 运行结果：

```text
my name ishka
try to learn 120天
```

- 代码可以正常运行，说明环境已经打通。
- 一个细节问题是第一行输出少了一个空格，更自然的写法应为：

```python
print(f"my name is {name}")
```

## 遇到的问题

| 问题 | 原因 | 解决方式 |
| --- | --- | --- |
| 输出 `my name ishka` 不够自然 | `f-string` 中 `is` 和 `{name}` 之间少了一个空格 | 修改为 `print(f"my name is {name}")` |
| 对 `pwd` 的含义不清楚 | 只记住了 `ls` 的用途，还没有建立常用终端命令的整体认知 | 补充记忆：`pwd = print working directory`，用于显示当前所在路径 |
