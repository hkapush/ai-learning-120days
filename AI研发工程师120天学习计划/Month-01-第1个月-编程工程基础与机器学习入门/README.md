# 第1个月-编程工程基础与机器学习入门

## 月度目标

补齐 AI 研发的工程底座，能用 Python 完成数据处理、机器学习建模和基础实验记录。

## 周主题

- 第 1 周：Python 进阶 / NumPy/Pandas / 可视化 / Git/Linux
- 第 2 周：算法与数据结构 / 复杂度 / LeetCode 高频题 / 代码规范
- 第 3 周：机器学习基础 / 回归/分类/聚类 / 模型评估 / 特征工程
- 第 4 周：完成第一个 ML 项目 / 实验记录 / 复盘与简历素材

## 月末检查

- 是否完成 30 天文件夹中的每日产出。
- 是否至少完成 1 个可展示项目或阶段性作品。
- 是否写出月度复盘，明确下个月补强点。
